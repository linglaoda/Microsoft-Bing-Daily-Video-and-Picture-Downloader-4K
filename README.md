# Microsoft_Bing_Daily_Video_-Image-_Downloader
微软Bing每日图片(视频)下载器 (最近30day)

## 该脚本允许您下载 30天 内的 Bing 每日图片(视频)
## 该脚本下载的文件清晰度为 4K (2160P)(2160*3840) 


## 🎀使用方法
##### Pip安装支持库后根据需要运行 main.py 或 only-video.py 即可
